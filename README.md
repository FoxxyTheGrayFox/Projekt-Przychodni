Oto prosty projekt w języku c# obsługujący przychodnię lekarską (z perspektywy pracowników recepcji). 
Główne funkcje tego programu to:
1. Możliwość dodania nowego pacjenta lub lekarza
2. Możliwość zarejestrowania nowej wizyty w przychodni
3. Możliwość wyświetlenia informacji o pacjencie oraz wizytach do niego przypisanych
4. Generacja wykresu z użyciem biblioteki ScottPlot - wykres pobiera dane statystyczne z ilości wizyt w danym miesiącu

Wszystkie dane o pacjentach, lekarzach oraz wizytach są przechowywane w pliku przychodnia.json, plik ten można znaleźć w folderze BIN, tam również znajduje się wygenerowany wykres.
